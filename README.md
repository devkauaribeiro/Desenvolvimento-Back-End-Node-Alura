# 🚀 Formação Desenvolvimento Backend Node.js - Alura

Bem-vindo(a) ao meu repositório de estudos e projetos da formação completa em **Desenvolvimento Backend com Node.js** pela [Alura](https://www.alura.com.br/carreiras/desenvolvimento-backend-nodejs). 📚

## 🎯 Objetivo

Este repositório documenta minha jornada desde os fundamentos do JavaScript até o desenvolvimento de arquiteturas complexas, microsserviços, DevOps e Cloud Computing. Ele serve como meu portfólio prático e central de consultas.

## 🛠️ Tecnologias e Ferramentas Abordadas

Devido à profundidade da formação, o ecossistema de tecnologias deste repositório inclui:

* **Linguagens:** JavaScript, TypeScript, Bash/Shell Script
* **Plataformas e Frameworks:** Node.js, Express.js, Nest.js, React
* **Bancos de Dados:** PostgreSQL, MySQL, SQLite, MongoDB, Redis, DynamoDB
* **ORMs e Ferramentas de Dados:** Sequelize, TypeORM, GraphQL
* **Arquitetura e Padrões:** POO, SOLID, Clean Architecture, DDD, Microsserviços, Mensageria, RPC, WebSockets (Socket.IO)
* **DevOps e Cloud (AWS):** Docker, Podman, Kubernetes, EC2, VPC, ECS, EKS, Terraform, Ansible, GitHub Actions (CI/CD)
* **Segurança e Testes:** Autenticação (JWT, OAuth 2.0), Secret Scanning, DAST, SAST, Testes Unitários e de Integração
