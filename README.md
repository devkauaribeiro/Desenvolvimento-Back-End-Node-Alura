# 🚀 Formação Desenvolvimento Front-End React - Alura

Bem-vindo(a) ao meu repositório de estudos e projetos da formação completa em **Desenvolvimento Front-End com React** pela [Alura](https://www.alura.com.br/carreiras/desenvolvimento-frontend-react). 📚

## 🎯 Objetivo

Este repositório documenta minha jornada desde os fundamentos do Front-End até o desenvolvimento de arquiteturas complexas com React, design systems, microfrontends e liderança técnica. Ele serve como meu portfólio prático e central de consultas.

## 🛠️ Tecnologias e Ferramentas Abordadas

* **Linguagens:** HTML, CSS, JavaScript, TypeScript
* **Frameworks e Bibliotecas:** React 19, Next.js, Remix, Radix UI, styled-components, Tailwind CSS, Redux, Context API
* **Ferramentas e Ecosistema:** Node.js, Vite, Turborepo, GraphQL, Socket.IO, Supabase, Nest.js (BFF)
* **Testes:** Jest, React Testing Library, Cypress
* **DevOps e CI/CD:** GitHub Actions, ESLint, Deploy contínuo
* **Observabilidade:** New Relic
* **Segurança:** OWASP, autenticação avançada
* **Arquitetura:** Atomic Design, Microfrontends, SOLID, Clean Architecture, Design Systems
